# 規則層級分類說明

## 規則層級架構（由高到低）

### 1. 最高法規

- **中華民國憲法**
- 所有規則的最高依據

### 2. 通用台灣法規

- **六法**（民法、刑法、行政法、商事法、民事訴訟法、刑事訴訟法）
- **專利權法**
- 適用於全台灣的通用法律

### 3. 主管機關法規

- **社會相關法規**（社會工作師法、社會救助法、人民團體法等）
- **文化相關法規**（文化資產保存法、文化創意產業發展法等）
- 主管機關制定的專門法規

### 4. 主規則

- **社區發展協會章程**
- **社區幹部選舉罷免相關規則**
- 組織內部最高層級的規則

### 5. 已定規則

- 已正式確定的系統規則
- 需電子簽章才能修改

### 6. 待訂規則

- 討論中但尚未正式確定的規則
- 可透過談話調整（開發階段）

### 7. 道德限制類

- 道德倫理層面的限制與規範
- 雖非法規但具有約束力

### 8. 精神理念類

- 系統的核心價值與理念
- 指導原則與哲學基礎
