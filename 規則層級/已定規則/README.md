# 已定規則

## 已正式確定的系統規則

### 閉環消費價值分配規則

- 50% 推動消費循環與反利生產消費者
- 50% 組織運作（40%人事、30%營運、20%發展、10%儲備）
- 動態分配機制（營業額增加，成本下降）
- 支出限制與無資本利得原則
- 科目白名單控制（非指定科目不得核銷）
- 直營商業本益比控制（P/E 比 > 50%，不含人事費）

### 系統架構規則

- 條約層、規則層、執行層三層架構
- 模組不能透過談話調整，需電子簽章
- 創辦人專利保護（眾利價值保護，否決停用權）
- 可究責權責分離（設計可究責 vs 營運可究責）

### PII 安全規則

- 物理隔絕保護（外接設備儲存 PII 對照表）
- 主機不定時呼叫機制（完全無規律可循）
- 三種訊號機制（兩種假訊號、一種真訊號，主動誘捕）
- 資料修改先儲存設備端，當日呼叫時一併送出
- 主機主動呼叫（僅限：家屬緊急求助、行政機關正式要求、幫主人找手機）
