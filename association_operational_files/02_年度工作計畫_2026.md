# 新北市三重區五常社區發展協會 113年度（2026年）工作計畫

**制定日期**：2026-01-15  
**計畫年度**：113年度（2026年1月1日至12月31日）  
**組織區域**：新北市三重區五常里、五順里、仁忠里

---

## 一、現況分析

### 1.1 社區基本資料
- **服務範圍**：五常里、五順里、仁忠里
- **總人口數**：約 9,000-10,000 人
- **總戶數**：約 3,800-4,000 戶
- **老年人口比例**：19.3%（接近超高齡社會門檻）
- **高等教育比例**：35.5%
- **平均屋齡**：34年

### 1.2 社區特色與挑戰
- **特色**：成熟型高密度住商混合區、高教育程度支持數位化
- **挑戰**：
  - 超高齡社會挑戰（19.3%老年人口）
  - 機車停車供需失衡（5,800輛 vs 60個公共車位）
  - 商業數位化程度為零，存在轉型機會
  - 新舊社區物業管理斷層

### 1.3 社區資源
- **地理優勢**：三重區通往台北市士林、北投地區的重要門戶
- **鄰近計畫**：銀新未來城、仁義重劃區
- **主要街道**：五華街（商業大動脈）、仁愛街（核心軸線）、正義北路（商業輻射）

---

## 二、年度工作目標

### 2.1 總體目標
1. 推動智慧社區服務平台，提升社區生活品質
2. 關懷高齡人口，推動在地安養與社區互助
3. 促進社區數位轉型，協助在地商家數位化
4. 改善社區環境，解決機車停車問題
5. 建立社區資源整合機制，提升服務效率

### 2.2 具體目標
- 完成智慧社區服務平台基礎建設
- 辦理高齡關懷活動至少 12 場次
- 辦理舞蹈運動課程（每週3日，全年156日，每日2小時）
- 協助至少 20 家在地商家導入數位服務
- 推動社區機車停車改善方案
- 建立社區資源資料庫與知識庫
- 完成wuchang.life智慧社區平台系統規劃與建置

## 三、工作項目與執行計畫

### 3.1 社區福利服務

#### 3.1.1 高齡關懷服務
- **目標**：因應 19.3% 老年人口，推動在地安養與社區互助
- **活動內容**：
  - 每月辦理高齡健康促進活動
  - 建立社區關懷據點
  - 推動志工服務網絡
  - 辦理長者數位學習課程
- **預期成果**：服務至少 200 人次高齡居民

#### 3.1.2 弱勢族群關懷
- **目標**：關懷社區內弱勢家庭與個人
- **活動內容**：
  - 辦理節慶關懷活動
  - 提供物資協助
  - 連結社會資源
- **預期成果**：服務至少 50 戶弱勢家庭

### 3.2 社區環境改善

#### 3.2.1 機車停車改善方案
- **目標**：改善 5,800 輛機車 vs 60 個公共車位的供需失衡
- **活動內容**：
  - 進行機車停車需求調查
  - 與相關單位協調增設停車空間
  - 推動停車秩序管理
- **預期成果**：提出具體改善方案並推動執行

#### 3.2.2 社區環境美化
- **目標**：改善社區環境品質
- **活動內容**：
  - 辦理社區清潔日活動
  - 推動綠美化工作
  - 改善公共空間
- **預期成果**：辦理至少 4 場次環境改善活動

### 3.3 社區文化活動

#### 3.3.1 社區文化保存
- **目標**：促進社區認同與文化傳承
- **活動內容**：
  - 辦理社區文化導覽
  - 記錄社區歷史與故事
  - 辦理文化節慶活動
- **預期成果**：辦理至少 6 場次文化活動

#### 3.3.2 社區交流活動
- **目標**：促進居民互動與社區凝聚力
- **活動內容**：
  - 辦理社區聯誼活動
  - 推動社區志工服務
  - 辦理社區成果展
- **預期成果**：辦理至少 8 場次交流活動

### 3.4 智慧社區服務

#### 3.4.1 智慧社區平台建置
- **目標**：建立智慧社區服務平台，提升服務效率
- **活動內容**：
  - 建置社區服務網站（wuchang.life）
  - 開發社區服務應用程式
  - 建立社區資源資料庫
- **預期成果**：完成平台基礎建設並上線運作

#### 3.4.2 數位轉型輔導
- **目標**：協助在地商家數位化轉型
- **活動內容**：
  - 辦理數位轉型說明會
  - 提供商家數位服務輔導
  - 建立社區電商平台
- **預期成果**：協助至少 20 家商家導入數位服務

## 四、預算編列

### 4.1 收入預算
| 項目 | 金額（新台幣） | 備註 |
|------|--------------|------|
| 會費收入 | 170,400 | 會員會費（每人每月100元，1月收取全年，150人×1,200元，退會不退費） |
| 講師回捐 | 265,200 | 舞蹈社講師回捐 |
| 其他收入 | 0 | 活動收入等 |
| **合計** | **1,085,600** | |

**備註**：
- 本年度資金來源除會費外，全部來自企業捐助
- 補助申請金額：0（本年度不申請政府補助）

### 4.2 支出預算
| 項目 | 金額（新台幣） | 備註 |
|------|--------------|------|
| 人事費 | 0 | 無人事成本（會務由五常物業規劃顧問股份有限公司承攬） |
| 業務費 | 1,011,500.0 | 活動執行費用（含舞蹈運動課程331,500元、會務承攬費300,000元，依協會盈餘計算，最高不超過700,000元） |
| 設備費 | 85,000 | 設備購置與維護 |
| 行政管理費 | 80,000 | 行政作業費用（定額） |
| 系統維護費 | 85,000 | 系統規劃、維護與升級（含wuchang.life平台） |
| **合計** | **1,261,500** | |

### 4.3 預算說明

**重要變更**：
- 會務承攬：依協會盈餘計算，最高不超過700,000元整（五常物業規劃顧問股份有限公司承攬會務，總幹事江政隆由公司聘用派駐本會，契約規範為協會盈餘金額，但不得超出700,000元整）
- 不足額補助：由上品食品行（統一編號：34778660）捐贈補足不足額部分
- 實際支出預算：1,261,500.0 元（含舞蹈運動課程331,500元、會務承攬費300,000元，依協會盈餘計算，最高不超過700,000元）
- 年終結餘預估：-175,900.0 元（需由上品食品行（統一編號：34778660）捐贈補足）
- 本預算需經會員大會通過
- 實際執行需依企業捐助到位情況調整
- 所有支出需符合協會章程與財務管理規範
- 收入與支出總額對應：850,000 元

## 七、風險管理

### 7.1 可能風險
- 會員數減少（轉型陣痛期）
- 企業捐助未到位
- 活動參與人數不足
- 經費執行進度落後
- 人力資源不足

### 7.2 因應措施
- 提前確認企業捐助意願
- 加強活動宣傳與推廣
- 建立進度追蹤機制
- 招募志工協助執行
