# 新北市三重區五常社區發展協會 法定文件產出時程表

**版本**：1.0  
**制定日期**：2026-01-17  
**依據法源**：人民團體法、社區發展工作要點、主管機關規範

---

## 一、年度文件（每年產出）

### 1.1 年度工作計畫
- **產出頻率**：每年1次
- **提交期限**：每年1月31日前
- **負責單位**：秘書處
- **文件格式**：年度工作計畫書
- **檔案範例**：`02_年度工作計畫_2026.md`

### 1.2 年度決算
- **產出頻率**：每年1次
- **提交期限**：每年3月31日前
- **負責單位**：財務組
- **文件格式**：年度決算書（含收支決算表、資產負債表、現金出納表、財產目錄、基金收支表）
- **檔案範例**：`財務_年度決算_YYYY.md`

### 1.3 年度預算
- **產出頻率**：每年1次
- **提交期限**：每年11月30日前
- **負責單位**：財務組
- **文件格式**：年度預算書（含收支預算表、員工待遇表）
- **檔案範例**：`財務_年度預算_YYYY.md`

### 1.4 年度工作報告
- **產出頻率**：每年1次
- **提交期限**：每年12月31日前
- **負責單位**：秘書處
- **文件格式**：年度工作報告
- **檔案範例**：`報告_年度工作報告_YYYY.md`

### 1.5 會員大會會議記錄
- **產出頻率**：每年1次（或依章程規定）
- **提交期限**：會議結束後15日內
- **負責單位**：秘書處
- **文件格式**：會員大會會議記錄
- **檔案範例**：`meetings/meeting_YYYYMMDD_會員大會.md`

## 九、文件管理建議

### 9.1 檔案命名規則
```
類別_文件名稱_日期或期間.md
```

範例：
- `財務_月報表_202601.md`
- `meetings/meeting_20260115_理監事會會議.md`
- `報告_季度工作進度_2026Q1.md`
- `補助_申請計畫書_社區發展工作補助_20260120.md`

### 9.2 檔案分類目錄
```
association_operational_files/
├── meetings/          # 會議記錄
├── financial/         # 財務文件
├── reports/           # 報告文件
├── activities/        # 活動文件
├── grants/            # 補助文件
└── organization/      # 組織文件
```

### 9.3 文件版本管理
- 所有文件需有版本號
- 文件修改需有修改記錄
- 重要文件需有審核流程
