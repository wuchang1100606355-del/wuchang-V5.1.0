## 一、會議演練系統

### 1.1 系統功能

`association_meeting_demo.py` 提供以下功能：

1. **模擬例行會議**：模擬理監事會會議流程
2. **生成會議記錄**：自動生成標準格式的會議記錄
3. **整合知識庫**：自動將會議記錄整合至知識庫
4. **更新行事曆**：可整合至行事曆系統

### 1.2 使用方式

```bash
python association_meeting_demo.py
```

## 二、會議記錄格式

### 2.1 標準格式

會議記錄包含以下章節：

1. **會議基本資訊**
   - 會議日期、時間、地點
   - 主持人、記錄人

2. **出席人員**
   - 理事會成員
   - 監事會成員
   - 工作人員

3. **議程**
   - 議程項目列表

4. **討論內容**
   - 各議程項目的討論內容
   - 發言人與發言內容

5. **決議事項**
   - 決議項目
   - 表決結果

6. **待辦事項**
   - 待辦項目
   - 負責單位
   - 完成期限
   - 狀態

7. **下次會議**
   - 時間、地點

8. **會議記錄確認**
   - 主持人簽名
   - 記錄人簽名

### 2.2 檔案命名規則

```
meeting_YYYYMMDD_會議類型.md
```

範例：`meeting_20260117_理監事會會議.md`

## 四、知識庫整合

### 4.1 自動整合

會議記錄生成後，系統會自動：

1. 將會議記錄儲存至 `association_operational_files/meetings/` 目錄
2. 更新知識庫，將會議記錄加入行政文件庫
3. 建立文件關聯關係

### 4.2 查詢方式

可透過以下方式查詢會議記錄：

```python
# 透過 API 查詢
GET /api/community/knowledge/search?q=會議記錄

# 透過 Python 程式查詢
from association_automation_system import AssociationAutomationSystem
system = AssociationAutomationSystem()
kb = system.load_knowledge_base()
```

## 五、系統整合

### 5.1 與自動化系統整合

會議記錄生成系統可與以下系統整合：

1. **行政文件管理系統**：會議記錄作為行政文件管理
2. **知識庫系統**：會議記錄整合至知識庫
3. **行事曆系統**：會議時程整合至行事曆
4. **待辦事項系統**：待辦事項自動建立追蹤

### 5.2 自動化流程

```
會議召開
    ↓
會議記錄生成（AI 輔助）
    ↓
會議記錄儲存
    ↓
知識庫更新
    ↓
待辦事項建立
    ↓
行事曆更新
```
