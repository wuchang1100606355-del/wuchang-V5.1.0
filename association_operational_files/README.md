## 三、使用方式

### 3.1 文件查詢
- 透過知識庫系統查詢：`/api/community/knowledge/search?q=...`
- 直接閱讀 Markdown 檔案
- 透過系統文件管理介面

### 3.2 文件更新
- 所有文件更新需有版本記錄
- 重要文件更新需經理監事會審核
- 文件更新需同步更新知識庫

- 系統會自動讀取這些文件
- AI 程序會參考這些文件生成新文件
- 自動化流程會依據這些文件執行
