# Google Tasks 伺服器端同步機制說明

## 同步機制

### 1. Google Tasks 雲端特性

Google Tasks 是雲端服務，所有任務都存儲在 Google 的伺服器上。當您在本地創建任務時：

- ✅ **任務立即存儲到 Google 雲端**
- ✅ **任何有權限的設備都可以訪問**
- ✅ **伺服器端可以通過 API 讀取**

### 3. 同步時間

| 階段 | 時間 | 說明 |
|------|------|------|
| 本地創建任務 | 立即 | 任務立即上傳到 Google 雲端 |
| 伺服器檢查 | 最多 60 秒 | 伺服器端下次輪詢時發現 |
| 總延遲 | 0-60 秒 | 取決於伺服器上次檢查的時間 |
