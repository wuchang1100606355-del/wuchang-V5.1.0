# 雙J重點記憶系統（完整版）

## 核心原則

**工作日誌、Google非營利規範、系統白皮書、雙J部署、雙J維運、最高權限授權、工具清單** 列為重點記憶，雙J工作前後**必讀、必寫、必查**。

系統白皮書所有功能優化可改，**不得降低標準**。

- 位置: dual_j_work_logs/

- 工作後：必寫本次記錄

- 工作前：必讀系統標準
- 優化時：必確認標準未降低

### 4. 雙J部署流程
- 位置: DUAL_J_FULL_DEPLOYMENT_STANDARD.md
- 工作前：必讀部署標準流程
- 部署時：必遵循標準步驟

### 5. 雙J維運流程
- 位置: DUAL_J_MAINTENANCE_STANDARD.md
- 工作前：必讀維運標準流程

### 6. 最高權限授權
