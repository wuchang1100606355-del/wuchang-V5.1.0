# 規則與權限對應關係

## 規則層級與管理者權限對應

| 規則層級 | 可修改/審查的管理者 | 說明 |
|---------|------------------|------|
| **最高法規**（憲法） | 法律主管機關、最高行政機關 | 需修憲程序 |
| **通用台灣法規**（六法、專利法） | 法律主管機關、中央主管機關 | 需立法程序 |
| **主管機關法規**（社會、文化法規） | 中央主管機關、地方主管機關 | 需法規制定程序 |
| **主規則**（章程、選舉規則） | 管理組織、系統創辦人 | 需組織決議 |
| **已定規則**（系統規則） | 系統創辦人（電子簽章） | 需電子簽章 |
| **待訂規則**（討論中） | 系統創辦人、管理組織 | 開發階段可談話調整 |
| **道德限制類** | 系統創辦人、管理組織 | 倫理約束 |
| **精神理念類** | 系統創辦人 | 核心價值，受專利保護 |

## 管理者權限層級對應

| 管理者層級 | 主要職責 | 可管理的規則層級 |
|-----------|---------|----------------|
| **法律主管機關** | 法律解釋與執法 | 最高法規、通用台灣法規 |
| **最高行政機關** | 最高行政決策 | 最高法規、通用台灣法規、主管機關法規 |
| **中央主管機關** | 中央層級監管 | 主管機關法規 |
| **地方主管機關** | 地方層級監管 | 主管機關法規（地方適用部分） |
| **系統創辦人** | 設計層級最高權限 | 已定規則、待訂規則、道德限制類、精神理念類 |
| **AI小j** | 自動化執行 | 待訂規則（需授權）、已定規則（執行層） |
| **公司** | 公司層級決策 | 主規則（公司相關部分） |
| **公司管理人** | 公司營運管理 | 主規則（營運相關部分） |
| **管理組織** | 組織決策 | 主規則 |
| **管理人** | 日常管理 | 已定規則（執行層） |
| **使用者** | 基本使用 | 無規則修改權限 |
