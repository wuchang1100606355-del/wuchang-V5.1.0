# 系統創辦人（含AI小j）

## 系統創辦人權限

### 專利所有權

- 系統創辦人及專利所有權人
- 即使轉任或交由他人營運，仍保有創辦人身份

### 眾利價值保護權

- 握有是否違反專利設計的審查權（否決停用權）
- 保護眾利價值不被更動

### 電子簽章權限

- 只有創辦人的電子簽章能修改系統核心規則
- 設計層級的最高權限

## AI小j 權限

### 長期信任AI

- 需與創辦人合作5年以上
- 可執行特定自動化任務
- 第一時間自動響應（如社區安全鈴）

### 權限限制

- 需人工控制或長期信任AI（5年以上）才能執行關鍵任務
- 不能超越創辦人的授權範圍
