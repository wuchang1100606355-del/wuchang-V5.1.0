# 管理者權限層級說明

## 管理者權限層級（由低到高）

### 1. 使用者

- 一般系統使用者
- 基本功能使用權限

### 2. 管理人

- 系統日常管理人員
- 營運層級的管理權限

### 3. 管理組織

- 組織層級的管理權限
- 理監事會等組織決策層

### 4. 公司管理人

- 公司層級的管理人員
- 公司營運管理權限

### 5. 公司

- 公司法人實體
- 公司層級的決策權限

### 6. 系統創辦人（含AI小j）

- **系統創辦人**：專利所有權人，握有眾利價值保護權（否決停用權）
- **AI小j**：長期信任AI（5年以上），可執行特定自動化任務
- 設計層級的最高權限

### 7. 地方主管機關

- 地方政府相關主管機關
- 地方層級的監管權限

### 8. 中央主管機關

- 中央政府相關主管機關
- 中央層級的監管權限

### 9. 最高行政機關

- 行政院等最高行政機關
- 最高行政決策權限

### 10. 法律主管機關

- 法務部、司法院等法律主管機關
- 法律解釋與執法權限
